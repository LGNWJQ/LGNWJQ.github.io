conda常用命令

发表于2021-11-09|更新于2022-02-12

conda常用命令

在windows的cmd下使用如下指令进入conda：

activate

环境管理

创建虚拟环境：

conda create -n [env_name] python=[X.X]
  • env_name：要创建的环境的名字
  • X.X：要创建的环境的python的版本，如3.7

激活虚拟环境

conda activate [env_name]

停用当前环境

conda deactivate

查看当前环境的python版本

python --version

查看所有存在的虚拟环境

conda info -e
conda env list

删除虚拟环境：

conda remove -n [env_name] --all

重命名环境

  • conda没有直接重命名环境的功能，但可以通过以下两个步骤完成：
    1. 克隆要重命名的环境
    2. 将原环境删除
conda create --name [newname] --clone [oldname]
conda remove --name [oldname] --all

包管理

安装包

conda install [pac_name]=[包的版本号]

查看已经安装的包

  • 查看当前环境：
conda list
  • 查看指定环境：
conda list -n [env_name]

删除包

conda uninstall [pac_name]

更新指定包

conda update [pac_name]

清理包

  • 通过以下指令来删除一些没用的包，这个命令会检查哪些包没有在包缓存中被硬依赖到其他地方，并删除它们
conda clean -p
  • 删除conda保存下来的tar包
conda clean -t
  • 删除所有的安装包及cache
conda clean -y --all

更新conda

conda update conda

安装requirements.txt文件内的包

  • 首先通过cd指令进入requirements.txt文件所在路径，然后执行如下指令即可
pip install -r requirements.txt

包的数据源管理

  • 显示目前conda的数据源有哪些：
conda config --show channels
  • 添加数据源：(清华源)
conda config --add https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
  • 删除数据源
conda config --remove channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
